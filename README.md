# 파일 관리기 테스트 버전

한양대학교 ERICA "데이터처리" 과목의 프로젝트 테스트버전 중 하나입니다.


## 1. 설치 및 실행

### 1.1 환경 설정

레포지토리 복사 후 다음 명령어로 필요한 라이브러리를 설치합니다.
```
pip install -r requirements.txt
```
```
pip install fastapi fitz numpy pandas Pillow scikit_learn sentence_transformers starlette uvicorn
```

### 1.2 서버 및 화면 실행
**백엔드** <br>
```main.py``` 를 실행하여 백엔드 서버를 실행합니다.  
로컬호스트 127.0.0.1:8000 포트에서 FastAPI 서버가 실행됩니다.

**프론트엔드** <br>
VS Code 환경의 **[Go Live](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)** (Live server 확장 프로그램)를 이용하거나 별도의 로컬 웹 서버를 구동하여 브라우저에서 실행합니다.  

<br>

## 2. 기본 사용법

1. 파일 선택: 중앙의 파일 혹은 폴더 아이콘을 클릭해 정리할 파일 또는 폴더를 선택합니다.
2. '다음 단계로'를 클릭하여 각 파일의 분석을 진행합니다. 그 다음 분류 단계까지 자동으로 진행됩니다.  
분류가 불가한 파일은 하단에 별도로 표시됩니다.
3. 오른쪽에서 특정 파이프라인에 따라 분류 결과를 확인하실 수 있습니다. **아직 파일이 실제로 이동된 상태가 아닙니다**.
4. 드래그 & 드롭으로 파일을 직접 분류할 수 있습니다. 폴더 또한 추가가 가능하며, 이전 단계에서 분류되지 않았던 파일들도 수동 분류 항목에 나타납니다. 직접 원하는 곳으로 이동하여 정리하거나, 그대로 두어 이동되지 않게 할 수 있습니다.
5. 아지막으로 분류 확정을 누르면 파일이 이동하게 됩니다.

## 3. 기타 사항

### 3.1 지원 확장자
* PDF(```.pdf```)
* 이미지(```.jpg```, ```.jpeg```, ```.png```, 등)
* CSV(```.csv```)

### 3.2 임베딩 모델
텍스트와 이미지 모두 ```clip-ViT-B-32```를 사용합니다.

### 3.3 분류 기준
현재는 임베딩 벡터 기준, Scikit-learn의 병합 군집화(```AgglomerativeClustering```) 알고리즘을 사용하여 코사인 거리 기준 유사도 0.2 이하를 임계값으로 설정하여 유사한 파일끼리 군집화합니다.  
분석에 실패했거나 군집 크기가 1인 동떨어진 파일들은 수동 분류 항목으로 이동됩니다.
